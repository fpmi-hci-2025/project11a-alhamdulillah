# Лабораторная работа №6: Оценка ранних концепций пользовательского интерфейса

## Предметная область

Проект "Махачкала" - сеть ресторанов с уникальным предложением "шаурма за 5 рублей". Система включает веб-приложение и мобильное приложение для клиентов, администраторов, курьеров и исполнителей заказов.

## Цель работы

Изучить методы оценки ранних концепций применительно к разработке пользовательского интерфейса приложения для сети ресторанов "Махачкала".
