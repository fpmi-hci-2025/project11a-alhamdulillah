# Лабораторная работа №7: Реализация базового функционала

## Цель работы

Разработка базового функционала приложения "Махачкала", включая серверное приложение, веб-интерфейс и мобильное приложение.

## Используемые технологии

**Backend**:
- Node.js + Express / Python + FastAPI
- PostgreSQL
- REST API

**Frontend (Web)**:
- React / Vue.js
- Bootstrap / Tailwind CSS
- Webpack / Vite

**Mobile**:
- React Native / Flutter
- Redux / MobX

**DevOps**:
- Docker
- GitHub Actions
- Jest / Pytest
