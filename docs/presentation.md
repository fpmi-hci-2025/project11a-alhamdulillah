# Презентация проекта

---

## 1. Название проекта

**Проект:** Информационная система сети ресторанов Махачкала  
**Команда:** project11a-alhamdulillah  
**Тип проекта:** Веб + мобильное приложение с серверной частью  

---

##  2. Цель и задачи проекта

### Цель проекта

Разработка информационной системы для управления сетью ресторанов, меню и позициями меню с поддержкой веб- и мобильных клиентов.

### Основные задачи

* анализ предметной области;
* проектирование архитектуры системы;
* разработка backend API;
* разработка web- и mobile-клиентов;
* проектирование базы данных;
* документирование и тестирование.

---

##  3. Описание системы

Система предоставляет следующие возможности:

* управление ресторанами;
* управление меню и категориями;
* просмотр меню пользователями;
* REST API для внешних клиентов;
* поддержка масштабирования и контейнеризации.

---

##  4. Архитектура проекта

Архитектура системы построена по многоуровневому принципу:

* клиентский уровень (Web, Mobile);
* сервер приложений (API);
* уровень бизнес-логики;
* уровень доступа к данным;
* база данных.

Для проектирования использовались UML- и C4-диаграммы.
